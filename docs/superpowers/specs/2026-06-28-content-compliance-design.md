# Parus 词库合规与敏感词过滤设计方案 (Content Compliance & Moderation Spec)

本方案旨在针对词库进行合规性审查（针对中国大陆上市及合规标准），对涉及中国台湾地区、历史称谓、领土主权相关的敏感定义和例句进行清洗与修正，确保词库符合出版及上架合规要求。

---

## 一、 敏感内容排查结果与修复方案

通过对 `dict_v2.db` 全库 30.5 万词条的扫描，共发现 **1 项释义词条缺陷**、**2 个敏感例句用例（共涉及4条例句）**，以及部分常规地理词汇表述不规范问题。具体修复方案如下：

### 1. 例句清洗 (Examples Table)
* **敏感用例 1（台湾是我最爱的国家）**：
  * **涉及词条**：`любимый` (Tatoeba), `тайвань` (Tatoeba)
  * **原内容**：`Тайвань — моя любимая страна. | 台灣是我最愛的國家。`
  * **修正方案**：
    * 俄文修改为：`Тайвань — мой любимый регион Китая.`
    * 中文修改为：`中国台湾地区是我最爱的地区。`

* **敏感用例 2（台湾是一个岛国）**：
  * **涉及词条**：`островной` (Tatoeba), `тайвань` (Tatoeba)
  * **原内容**：`Тайвань — островное государство. | 台灣是一個島國。`
  * **修正方案**：
    * 俄文修改为：`Тайвань — островной регион Китая.`
    * 中文修改为：`中国台湾地区是一个岛屿地区。`

---

### 2. 释义校正 (Definitions Table)
为了完全符合国家和地区表述规范，对台湾相关词汇进行归一化表述修饰：

* **中华民国**：
  * **涉及词条**：`китайская республика` (BKRS)
  * **原内容**：`中华民国 zhōnghuá mínguó`
  * **修正方案**：修改为：`中华民国（历史称谓，1912-1949）`

* **台湾**：
  * **涉及词条**：`тайвань` (BKRS)
  * **原内容**：`台湾 táiwān`
  * **修正方案**：修改为：`中国台湾地区`

* **台湾人 / 台湾女人**：
  * **涉及词条**：`тайванец` (BKRS), `тайванька` (BKRS)
  * **原内容**：`台湾人` / `台湾[女]人`
  * **修正方案**：修改为：`中国台湾地区居民` / `中国台湾地区女性居民`

* **台湾的 / 台湾自行车节 / 台湾纪录片界**：
  * **涉及词条**：`тайваньский` (BKRS), `велофестиваль` (BKRS), `кинодокументалистика` (BKRS)
  * **原内容**：`台湾的` / `台湾自行车节` / `台湾纪录片界`
  * **修正方案**：前置修饰改为 `中国台湾地区的` / `中国台湾自行车节` / `中国台湾纪录片界`

## 二、 实施方式设计

本着数据驱动和“不手动改库、保证管线可重入”的原则，我们不直接用 SQL 执行一次性修改，而是在数据库构建流水线（`pipeline/phase3_build_db.py`）中增加一个**合规过滤器（Compliance Filter）**。

### 1. 流水线拦截逻辑
在 `phase3_build_db.py` 写入数据库前，增加如下两个拦截处理函数：

```python
def filter_compliance_definition(lemma, definition):
    # 1. 拦截中华民国
    if lemma == "китайская республика":
        return "中华民国（历史称谓，1912-1949）"
    # 2. 台湾词条规范化
    if lemma == "тайвань" and definition == "台湾 táiwān":
        return "中国台湾地区 táiwān"
    if lemma == "тайванец" and definition == "台湾人 táiwānrén":
        return "中国台湾地区居民 táiwānrén"
    if lemma == "тайванька" and definition == "台湾[女]人":
        return "中国台湾地区女性居民"
    
    # 3. 常规“台湾”替换为“中国台湾”
    # 注意：防止重复替换，先判断是否已经有“中国台湾”
    if "台湾" in definition and "中国台湾" not in definition:
        definition = definition.replace("台湾", "中国台湾")
    return definition

def filter_compliance_example(lemma, sentence_ru, sentence_zh):
    # 1. 过滤“最爱的国家”
    if "Тайвань — моя любимая страна" in sentence_ru or "台灣是我最愛的國家" in sentence_zh:
        return "Тайвань — мой любимый регион Китая.", "中国台湾地区是我最爱的地区。"
    # 2. 过滤“岛国”
    if "Тайвань — островное государство" in sentence_ru or "台灣是一個島國" in sentence_zh:
        return "Тайвань — островной регион Китая.", "中国台湾地区是一个岛屿地区。"
    
    # 3. 常规例句中文“台湾”替换为“中国台湾”
    if "台湾" in sentence_zh and "中国台湾" not in sentence_zh:
        sentence_zh = sentence_zh.replace("台湾", "中国台湾")
    if "台灣" in sentence_zh and "中国台湾" not in sentence_zh:
        sentence_zh = sentence_zh.replace("台灣", "中国台湾")
        
    return sentence_ru, sentence_zh
```

### 2. 为什么在 pipeline 层做？
- **100% 可复现**：后续任何词库更新（例如重新跑 DeepSeek 或引入新媒体语料），都不需要重新手动改库，构建脚本会自动完成合规重写。
- **杜绝二次污染**：无论数据源怎么变，流水线是最后一道关卡，可以保证最终导出的 `dict_v2.db` 永远是合规的。

## 三、 验证方案

1. **自动单元测试**：在构建脚本执行结束后，自动发起对 `dict_v2.db` 的查询：
   - 确保 `SELECT count(*) FROM examples WHERE sentence_zh LIKE '%岛国%' AND sentence_zh LIKE '%台湾%'` 的返回结果为 `0`。
   - 确保 `SELECT count(*) FROM definitions WHERE definition = '中华民国 zhōnghuá mínguó'` 的返回结果为 `0`。
2. **人工抽检**：打印出最终所有包含 `台湾` 关键字的释义和例句，以确认最终修正字面无误。
